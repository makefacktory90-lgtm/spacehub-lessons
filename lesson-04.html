Тема 01 00:00

Генерация изображений: Nano Banana, GPT Image

Денис показывает, как подключить инструмент генерации изображений к ассистенту. Разбираются доступные модели (Nano Banana, Nano Banana Pro, GPT Image, SiDream 4.5), стоимость каждой, работа с русским текстом. Ира объясняет, как фирменный стиль в промпте ускоряет создание каруселей.

  • Nano Banana Pro: лучшее качество на русском, стоимость 50 кредитов за картинку
  • GPT Image: хорош для общих задач, но хуже с русским текстом
  • Стиль в промпте инструмента = все картинки бота в едином визуале
  • Голосовые команды тоже работают: говорите боту что нарисовать, картинка летит в Telegram
  • Генерация стоит ~2 рубля за картинку на Nano Banana
Тема 02 10:10

Генерация видео: Veo, первый и последний кадр

Подключение инструмента генерации видео. Обзор моделей: Veo (Google), SiDream. Функция первого и последнего кадра позволяет создавать переходы между двумя изображениями. Обсуждение стоимости и реалистичных ожиданий от генерации.

  • Veo (Google): дешёвая и быстрая модель для видео, поддерживает вертикальный и горизонтальный формат
  • Первый и последний кадр: задаёте два изображения, модель генерирует переход
  • Стоимость видео: ~150 кредитов (~30 руб.) за 5 секунд
  • Ира: профессионалы делают тысячи генераций для хорошего кадра, это нормально
  • Формат (горизонтальный/вертикальный) указывается в промпте
Тема 03 21:12

Несколько генераторов в одном боте

Как подключить несколько инструментов генерации к одному ассистенту и переключаться между ними словами. Обновления интерфейса: появились чаты с тредами внутри AI from Space. Агентная система помнит контекст предыдущих запросов.

  • В одном боте могут быть Nano Banana + GPT Image + Veo одновременно
  • Ассистент сам выбирает нужный инструмент по вашей просьбе: «сделай с помощью Nano Banana»
  • Это не оркестратор: просто несколько инструментов у одного ассистента
  • Агентная система: бот помнит, что только что делал, можно просить доработать
  • Новый ChatGPT-like интерфейс: треды, история, удобнее для работы
Тема 04 29:26

Навыки (Skills): экономия токенов и цепочки

Новая функция, вдохновлённая Claude Code. Навык — это мини-программа с инструкциями, которая подгружается только когда нужна. Два режима: «по требованию» (экономит токены) и «всегда включен» (как постоянная память). Можно строить последовательные цепочки навыков.

  • Навык = инструкция, которая подгружается по требованию, а не висит постоянно
  • Экономия: вместо 10 000 токенов в памяти — одно предложение-описание
  • Context rot: чем больше токенов у модели, тем хуже она соображает
  • Цепочки: навыки можно выстраивать последовательно (поиск → анализ → текст → проверка)
  • Два режима: «по требованию» (90% случаев) и «всегда включен» (как факты)
  • Ира: это путь к полной автоматизации — робот идёт по чёткому маршруту
Тема 05 45:25

Подписки и монетизация бота

Полный разбор системы подписок: создание тарифных планов, оплата звёздами Telegram или рублями, расчёт маржинальности, автопродление. Как определить цену подписки исходя из стоимости кредитов.

  • Оплата: звёзды Telegram (сразу к вам) или рубли (через Дениса, раз в месяц)
  • Расчёт маржи: 1000 руб. = 5000 кредитов. Хотите x5 → даёте 1000 кредитов за 1000 руб.
  • Для 100 руб./мес: ~100 кредитов пользователю (маржа x5)
  • Несколько ассистентов на одну подписку — можно продавать пакетом
  • Автопродление работает только для месячных подписок
  • Промпт при покупке: бот сам здоровается с новым подписчиком
  • Лимиты: переопределяют дефолтные по сообщениям и/или кредитам
Тема 06 1:05:17

Роли: admin, user и наследование

Система ролей позволяет ассистенту вести себя по-разному с разными людьми. Роль admin наследует от user, но может переопределять настройки, память и инструменты. Можно ограничить функции для бесплатных пользователей и расширить для платных.

  • User — роль по умолчанию для всех. Не удалять!
  • Admin наследует от User, но может переопределять: настройки, память, инструменты
  • Память по ролям: User видит «ты работаешь на Дениса», Admin видит «я тебя создал»
  • Инструменты по ролям: бесплатным — базовые, платным — все генераторы
  • Подписка может переопределять роль: купил → стал Admin с расширенными правами
  • Это основа для администратора: бот отвечает клиентам, а вам пишет уведомления
Тема 07 1:18:27

Q&A: вопросы участников

Разбор вопросов: передача ассистентов между аккаунтами, экспорт в PDF через Code Interpreter, работа с большими базами знаний (100+ курсов), белорусские карты и международные платежи. Анонс субботы: глубокое погружение в память.

  • Передача ассистента: написать Денису, он перенесёт. Скоро — через интерфейс
  • Кнопка «Скопировать ассистента» — дублирует все настройки, память, инструменты
  • Экспорт в PDF: включить Code Interpreter, попросить ассистента сгенерировать файл
  • Большие базы знаний: информации может быть сколько угодно, вызывается по необходимости
  • Международные платежи (евро/доллары): в разработке, скоро
  • Суббота: детальный разбор памяти — стейджинг, загрузка файлов, отображение
Проверь себя

Квиз по уроку

1 Какая модель лучше всего генерирует картинки с русским текстом?

Nano Banana Pro лучше всех работает с русским текстом на изображениях. GPT Image тоже неплох, но по русскому тексту уступает. Стоимость Nano Banana Pro — 50 кредитов за картинку.

2 Что такое навык (Skill) в AI from Space?

Навык — это набор инструкций, который подгружается только когда нужен. Вместо огромного промпта в постоянной памяти (тратит токены всегда), навык активируется по триггеру. Это экономит токены и борется с context rot.

3 Что такое context rot?

Context rot — «гниение контекста». Чем больше токенов модель обрабатывает одновременно, тем хуже она соображает. Как если бы вы перед экзаменом за 5 минут прочитали всю книжку — ничего не запомните. Навыки решают эту проблему.

4 Как рассчитать маржинальность подписки на бота?

Пример: 1000 руб. = 5000 кредитов по себестоимости. Хотите маржу x5? Даёте пользователю 1000 кредитов за 1000 руб. Для подписки 100 руб./мес — это ~100 кредитов. Чем больше пакет, тем дешевле кредит для пользователя.

5 Что произойдёт, если подписка переопределяет роль?

При покупке подписки с переопределением роли пользователь автоматически получает все настройки новой роли: расширенные инструменты, больше лимитов, другой уровень размышления модели, дополнительную память. Пример: бесплатный user → купил подписку → стал admin с доступом к Nano Banana Pro и Veo.

6 Зачем нужны два режима активации навыка?

«По требованию» подгружает инструкцию только когда нужно (90% случаев). «Всегда включен» работает как постоянная память — для правил поведения, которые нужны практически в каждом разговоре. Разница между «всегда включённым» навыком и постоянной памятью: в памяти факты, в навыке — алгоритмы работы.

Практика

Домашнее задание

1 Подключите генерацию изображений

Откройте вашего ассистента в AI from Space. Добавьте инструмент генерации изображений, выберите модель Nano Banana (для экономии) или Nano Banana Pro (для качества). В промпт инструмента добавьте свой визуальный стиль. Попросите бота сгенерировать 3 картинки в этом стиле и убедитесь, что результат стабильный.

2 Создайте навык (Skill)

Зайдите в раздел навыков ассистента. Создайте навык «по требованию» для задачи, которую вы делаете периодически: например, генерация видео с определёнными параметрами, или создание текста по шаблону. Напишите чёткое описание навыка (одно предложение) и инструкции внутри. Протестируйте: попросите бота выполнить задачу и проверьте, что навык подгрузился.

3 Создайте подписку и посчитайте маржу

Зайдите в раздел «Подписки» вашего ассистента. Создайте тестовую подписку: (а) придумайте название, (б) поставьте цену в 1 звезду или 100 рублей, (в) рассчитайте лимит кредитов с маржой x5, (г) добавьте ассистента. Попробуйте купить свою подписку через Telegram и проверьте, что лимиты работают.

4 Настройте роль Admin

Создайте роль Admin (наследник User). Назначьте себя админом через раздел «Чаты по ролям». Для роли Admin: (а) добавьте память «я тебя создал», (б) повысьте уровень размышления до High, (в) включите инструмент «Написать человеку». Для User — выключите этот инструмент. Проверьте, что при общении от имени Admin вы получаете расширенный функционал.
