Тема 01 00:00

Введение: зачем нужны AI-ассистенты

Ира объясняет, почему AI-ассистенты — самый доступный вход в тему ИИ для не-технарей. Платформа Дениса — самый простой способ создать своего бота. Тренд на ассистентов очевиден: без них скоро будет сложно. Ира собирает ботов за 5 минут, и участники смогут делать то же самое.

  • AI-ассистенты — тренд, без которого скоро не обойтись
  • Платформа AI from Space — самый простой вход для не-технарей
  • Когда понимаешь механику, создание нового бота занимает 5-7 минут
  • Не нужно уметь кодить — всё работает через конструктор
Тема 02 05:51

Знакомство с платформой и создание ассистента

Денис показывает интерфейс AI from Space: регистрация, бонусные 2500 кредитов, создание первого ассистента. Встроенный помощник сам генерирует системное сообщение (промт) на основе наших ответов: какую роль выполняет бот, в каком тоне общается.

  • При регистрации начисляются 2500 бонусных кредитов
  • Встроенный ассистент помогает написать системное сообщение
  • Системное сообщение — это роль, задачи и тон общения бота
  • Всё работает через визуальный конструктор без кода
Тема 03 14:27

Основные настройки: модель, размышление, креативность

Разбираем ключевые настройки ассистента: выбор AI-модели (доступно 462 модели), уровень размышления (влияет на качество и стоимость ответов) и креативность (насколько вариативны ответы). По умолчанию стоит Gemini 3 Flash Preview — умная и недорогая.

  • 462 модели доступно, по умолчанию — Gemini 3 Flash Preview
  • Уровень размышления: выше — качественнее, но дороже
  • Креативность: средняя подходит для большинства задач
  • Стоимость моделей различается в разы — смотрите на цену
Тема 04 22:06

Система памяти ассистента

Денис показывает два способа дать ассистенту информацию: через системное сообщение (неудобно) и через раздел «Память» (правильный способ). Ячейки памяти можно включать, выключать и удалять, не трогая основной промт. Это удобнее и профессиональнее.

  • Раздел «Память» — удобный способ хранить информацию для бота
  • Ячейки памяти можно включать и выключать без правки промта
  • Не нужно засорять системное сообщение деталями
  • Бот использует ячейки памяти при каждом ответе
Тема 05 28:01

Инструменты: поиск в интернете и генерация картинок

Ключевая тема — инструменты ассистента. LLM сама по себе не умеет ходить в интернет или генерировать картинки. Инструменты дают ей эти возможности. Подключаем поиск в интернете (работает через Perplexity) и генерацию изображений (Nano Banana). Каждый инструмент — это дополнительные расходы кредитов.

  • Без инструментов LLM — просто 'застывший камень'
  • Поиск в интернете работает через Perplexity
  • Можно ограничивать поиск конкретными сайтами
  • Генерация картинок через Nano Banana — около 2 рублей за изображение
Тема 06 39:22

Как LLM работает под капотом

Денис показывает раздел 'Чаты' — полностью прозрачную историю работы ассистента. Каждый ответ можно раскрыть и увидеть: какая модель использовалась, какие инструменты вызывались, сколько кредитов потрачено. LLM ничего не помнит — каждый раз ей отправляется вся история переписки заново.

  • LLM ничего не помнит — каждый раз получает всю историю заново
  • Платформа полностью прозрачна — видно все 'под капотом'
  • Вызов инструмента удваивает стоимость запроса
  • Один запрос с поиском ≈ 2 кредита (≈30 копеек)
Тема 07 51:46

Вывод бота в Telegram через BotFather

Практический шаг: выводим созданного ассистента в Telegram. Заходим в BotFather, создаём нового бота, копируем токен и вставляем его в AI from Space. Бот сразу начинает работать в Telegram с кнопками, голосовым вводом и всеми настроенными инструментами.

  • BotFather — официальный бот Telegram для создания ботов
  • Имя бота должно заканчиваться на 'bot' и быть уникальным
  • Копируем токен из BotFather → вставляем в AI from Space
  • Аватарку бота ставим через настройки Telegram, не через платформу
Проверь себя

Квиз по уроку

1 Какая модель стоит по умолчанию при создании ассистента на AI from Space?

По умолчанию стоит Gemini 3 Flash Preview — она и умная, и достаточно дешёвая. Денис рекомендует начинать именно с неё.

2 Как правильно добавить информацию о себе для ассистента?

Раздел «Память» — самый удобный способ. Ячейки можно включать, выключать и удалять, не трогая системное сообщение.

3 Почему LLM без инструментов не может сказать, какая сегодня погода?

LLM — «неизменяемая коробочка», натренированная на данных прошлого. Без инструмента поиска в интернете она не знает актуальную информацию.

4 Что нужно скопировать из BotFather для подключения бота к AI from Space?

Из BotFather нужно скопировать токен бота и вставить его в настройки Telegram на платформе AI from Space.

5 Почему запрос с вызовом инструмента стоит дороже обычного?

При вызове инструмента модель обращается дважды: сначала решает вызвать инструмент, потом обрабатывает результат и отвечает пользователю.

6 Как участники придумали бота-факт-чекера на уроке?

Идея: создать основного ассистента, который спрашивает у трёх других с разными моделями и сравнивает ответы, снижая процент ложной информации.

Инструменты Возможности, которые дают ассистенту действовать: ходить в интернет, генерировать картинки, музыку и т.д.
Gemini 3 Flash Preview Рекомендованная модель по умолчанию — умная и недорогая, от Google
Промт (prompt) Текст запроса или инструкции для нейросети. Чем точнее промт, тем точнее ответ
Память ассистента Раздел платформы для хранения информации (имя владельца, задачи, контекст) в виде отдельных переключаемых ячеек
BotFather Официальный бот Telegram для создания и управления ботами. Через него получаем токен для связки с AI from Space
Уровень размышления Настройка, определяющая глубину 'думания' модели перед ответом. Выше — качественнее, но дороже
Креативность (temperature) Параметр, отвечающий за вариативность ответов: низкая — строгие, высокая — творческие и непредсказуемые
Perplexity AI-сервис поиска в интернете, используемый инструментом 'Поиск в интернете' на платформе AI from Space
Nano Banana Модель для генерации изображений, одна из самых доступных (~2 рубля за картинку)
Контекстное окно Объём информации (в токенах), который модель может обработать за один запрос. У Gemini — более 1 млн токенов
Факт-чекер Идея бота, придуманная на уроке: три ассистента с разными моделями сверяют информацию для снижения ошибок
